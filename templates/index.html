Derangement (Hat-Check) Problem

A derangement problem is also known as a hat-check problem in probabilistic theory. This problem involves assigning hats to people such that no one receives their own hat. This project is dedicated to solving the problem and presenting its concepts in a comprehensible manner.

We leverage mathematical equations and dynamic programming to efficiently compute the derangements for a given number of people.

Function Explanation

The function derangement(n) calculates the number of derangements (or permutations of the elements of a set) in which none of the elements appear in their original position. The recursive formula for the function is:

def derangement(n):
    if n == 0:
        return 1
    if n == 1:
        return 0
    return (n - 1) * (derangement(n - 1) + derangement(n - 2))
